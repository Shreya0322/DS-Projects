# DS-Projects

## Overview
This repository contains a collection of Data Structures projects developed in C to strengthen my understanding of fundamental data structures and algorithms. The projects focus on implementing various data structures from scratch and applying them to solve practical problems.

## Concepts Covered
- Linked Lists
- Hash Tables
- Searching Algorithms
- Dynamic Memory Allocation
- File Handling

## Project Structure
Each project is organized into a separate folder and includes the source code along with a README file explaining the project's objective, implementation, and execution steps.

## Requirements
- GCC Compiler
- Linux or Windows
- Command Line/Terminal
